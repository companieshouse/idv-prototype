What you’ll need to do as a Companies House authorised agent

Once registered, you must provide details about any changes to the authorised agent's Anti-Money Laundering (AML) registration. This includes if the:

  • agent is no longer registered with an AML supervisory body
  • AML registration details change, for example if the agent registers with a different AML supervisory body
