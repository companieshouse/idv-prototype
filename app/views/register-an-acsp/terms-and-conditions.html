What you’ll need to do as an ACSP

Once registered, you must provide details about any changes to the ACSP’s Anti-Money Laundering (AML) registration. This includes if the:

  • ACSP is no longer registered with an AML supervisory body
  • AML registration details change, for example if the ACSP registers with a different AML supervisory body

If you are no longer the person who is responsible for AML compliance at the ACSP, the ACSP must give details for a new relevant officer.

Under AML regulations, all ACSPs must keep records on any identity verification checks they complete. We may ask for more information on individual checks if we need to.
