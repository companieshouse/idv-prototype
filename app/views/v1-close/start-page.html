Close the Companies House authorised agent

If you close the authorised agent, MORRIS ACCOUNTING LTD will no longer be able to file with Companies House on behalf of clients.

If there are other users who have been added to the account, you should make sure they know they will no longer be able to access the account, or services for the authorised agent.
