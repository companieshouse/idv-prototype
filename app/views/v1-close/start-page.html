Close a Companies House authorised agent

Use this service to tell us if MORRIS ACCOUNTING LTD no longer needs to be a Companies House authorised agent. This is also known as an Authorised Corporate Service Provider (ACSP).

Before you start

If there are other users who have been added to the account, you should make sure they know they will no longer be able to access the account or services for the authorised agent.

What you'll need to do

We’ll tell you what happens when you close the agent and ask you to complete a declaration to confirm you understand. Then you’ll need to continue to confirm you want to close it.

This will permanently close the authorised agent. You only should continue if you are sure that MORRIS ACCOUNTING LTD no longer needs to file with Companies House on behalf of clients.

It takes around 5 minutes to use this service.
