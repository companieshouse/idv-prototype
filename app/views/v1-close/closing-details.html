MORRIS ACCOUNTING LTD

What will happen when you close the authorised agent account

MORRIS ACCOUNTING LTD will no longer be an authorised agent. This is also known as an Authorised Corporate Service Provider (ACSP).

You will not be able to use services for authorised agents

MORRIS ACCOUNTING LTD will no longer be able to:

  • verify people's identities for Companies House
  • file on behalf of clients

We'll publish information to confirm

We’ll publish the:

  • authorised agent’s name
  • status, to show it is no longer an authorised agent
  • date it stopped being an authorised agent
