View and update the authorised agent's details

Use this service to view and update details for JAMES MORRIS ACCOUNTING LTD.

{{ govukWarningText({ text: "If any of the details we hold about the authorised agent change, you must update them within 14 days", iconFallbackText: "Warning" }) }}

You’ll need to tell us if there are any changes to the:

  • agent's Anti-Money Laundering (AML) registration details
  • details we hold about the business

Updating your AML registration details

You need to update the AML registration details if the agent has registered with a different AML supervisory body, or the AML membership number changes.

The agent must always be registered with at least one AML supervisory body.

Adding details for a new registration

If the agent has registered with a new AML supervisory body, you’ll need to tell us the:

  • name of the AML supervisory body
  • AML membership number or ID

Updating details for an existing registration

You cannot edit the details for an existing registration. If there has been a change, for example, the agent has a new membership number or ID from the supervisory body, you will need to add this as a new registration.

You’ll need to delete the previous details once you’ve added the new ones.

It takes around 5 minutes to use this service.
